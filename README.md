# CodeVault Project Guide

## How to Run Modules

### Authentication Portal (Login Page)
Navigate to the `auth-portal` folder:
```bash
cd auth-portal
pnpm dev
```

### Chrome Extension
Navigate to the `code-vault` folder inside `chrome-extension`:
```bash
cd chrome-extension/code-vault
pnpm dev
```

### Backend Server
Navigate to the `server` folder:
```bash
cd server
pnpm run start:dev
```
